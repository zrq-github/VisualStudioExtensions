# VisualStudioExtensions
VS扩展. 目前已经将Leetcode通过WebViewe2整合进了插件中

## LeetCodeVsExtension 

使用WebView2, 将LeetCode的界面内置到 Visual Studio 中, 实现更加方便刷取每日一题的内容

- [ ] 快捷打开每日一题
- [ ] 在VS中写好代码, 选中指定代码, 右键/快捷键自动上传提交
